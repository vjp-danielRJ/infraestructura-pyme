# Análisis de Requisitos

## Descripción del cliente
Pequeña empresa que necesita presencia web y sistema de gestión interna.

## Requisitos funcionales
- Servidor web con Apache y PHP
- Base de datos MySQL/MariaDB
- Acceso remoto seguro mediante SSH
- Monitorización del sistema
- Copias de seguridad automáticas

## Requisitos no funcionales
- Alta disponibilidad
- Seguridad básica con firewall UFW
- Documentación completa para el cliente
